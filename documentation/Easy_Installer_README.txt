IACLS Time Tracker - Easy Installer
===================================

For users who want the simplest possible installation:

1. Double-click "Easy_Installer.app"
2. Follow the prompts
3. Enter your password when asked (for Homebrew installation)
4. Wait for installation to complete
5. Launch the app when prompted

What this installer does:
- Checks if Homebrew is installed
- Installs Homebrew if needed (requires password)
- Adds the IACLS tap to Homebrew
- Installs IACLS Time Tracker
- Offers to launch the app

Requirements:
- macOS 10.15 or later
- Internet connection
- Administrator password (for Homebrew)

If you prefer manual installation:
- Use Homebrew: brew tap markwbennett/iacls && brew install --cask iacls-time-tracker
- Or download from: https://github.com/markwbennett/TimeTrackButton/releases
