IACLS Time Tracker - macOS Distribution
=====================================

Contents:
- IACLS Time Tracker.app - Main application
- SketchyBar_Plugins/ - SketchyBar integration files
- setup_timetracker.sh - Automated setup script
- Documentation/ - Installation guides and documentation

Quick Start:
1. Read Documentation/macOS_Installation_Guide.pdf
2. Run setup_timetracker.sh for automated setup
3. Launch IACLS Time Tracker.app

For manual setup, see the documentation folder.
